# Spotify AutoFollow

Segue automaticamente todos os artistas das músicas curtidas.

## Como usar

### 1. Obtenha:
- CLIENT_ID
- CLIENT_SECRET
- REFRESH_TOKEN

### 2. Coloque no GitHub:
Em "Settings" → "Secrets and variables" → "Actions" → "New repository secret":

- CLIENT_ID
- CLIENT_SECRET
- REFRESH_TOKEN

### 3. Pronto: o script roda automaticamente a cada 6 horas.
